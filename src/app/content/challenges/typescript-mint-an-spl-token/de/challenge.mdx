# SPL-Token prägen

![Mint an SPL Token Challenge](/graphics/challenge-banners/typescript-mint-an-spl-token.png)

Das SPL-Token-Programm, insbesondere das Prägen von Tokens mit TypeScript, sollte der erste Ausgangspunkt auf deinem Weg als Solana-Entwickler sein.

Wenn du neu im SPL-Token-Programm bist, folge zuerst [diesem Kurs](/de/courses/spl-token-with-web3js)!

In dieser Challenge werden wir vier einfache Anweisungen implementieren:

- **Mint erstellen**: Erstelle einen `Mint Account` mit der Roh-Anweisung oder der abstrahierten Anweisung, die das SDK bietet.
- **Mint initialisieren**: Initialisiere den erstellten `Mint Account` mit der Roh-Anweisung oder der abstrahierten Anweisung, die das SDK bietet. *Der Mint sollte 6 Dezimalstellen haben, wobei die Mint-Autorität auf das Wallet gesetzt ist, das den Code ausführt, und keine Freeze-Autorität hat*.
- **Associated Token Account erstellen**: Erstelle und initialisiere ein `Associated Token Account` mit der Roh-Anweisung oder der abstrahierten Anweisung, die das SDK bietet.
- **21 Millionen Tokens prägen**: Präge 21 Millionen (21.000.000) der neu erstellten Tokens auf das neu erstellte `Associated Token Account`.

**Hinweis**: Wenn du nicht mit der Verwendung des SPL-Token-Programms mit TypeScript vertraut bist, solltest du zunächst [SPL Token mit Web3.js](/de/courses/spl-token-with-web3js) lesen, um dich mit den SDKs vertraut zu machen, die wir in dieser Challenge benötigen werden.
